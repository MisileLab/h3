## 문서 요약

"ell"이라는 Bash로 작성된 명령줄 인터페이스(LCLI)에 대한 문서입니다.\
주요 기능은 터미널에서 LLM(대형 언어 모델)에게 질문을 하고, 셸의 컨텍스트를 LLM에 전달하며, 대화할 수 있는 능력을 포함합니다.

**설치 및 사용법:**
- `git clone`으로 설치하며, 경로 설정이 필요합니다.
- `.ellrc` 파일에서 API 키와 모델 정보를 설정합니다.
- 질문 및 파일 입력을 통해 명령어 실행과 인터랙티브 모드 사용이 가능합니다.

**기타 정보:**
- FAQ에서는 "ell"의 네이밍, Bash 사용 이유, 유사 프로젝트와의 차별점 등을 논의합니다.
- MIT 라이선스 하에 기여를 환영합니다.

결국, "ell"은 LLM을 위한 간단하고 경량화된 터미널 도구입니다.
