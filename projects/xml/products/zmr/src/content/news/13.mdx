아래 글은 [Anthropic](https://anthropic.com)의 claude 3.5 sonnet으로 요약되었습니다.

1. 소개
- HTML5는 1990년대 초부터 계속 발전해온 HTML의 최신 버전입니다.
- HTML5는 이전 버전과의 호환성을 유지하면서 새로운 기능을 추가했습니다.

2. 구문
- HTML5는 HTML 구문과 XML 구문을 모두 지원합니다.
- 문서 인코딩 선언 방식이 간소화되었습니다.
- DOCTYPE 선언이 간단해졌습니다: `<!DOCTYPE html>`
- MathML과 SVG 요소를 HTML 문서 내에서 사용할 수 있습니다.

3. 언어
- 새로운 요소들이 추가되었습니다 (예: section, article, nav, header, footer, aside, figure 등).
- 새로운 폼 입력 타입이 추가되었습니다 (예: date, time, email, url 등).
- 새로운 속성들이 추가되었습니다 (예: contenteditable, hidden, data-* 등).
- 일부 요소와 속성의 의미가 변경되었습니다.
- 일부 요소와 속성이 더 이상 사용되지 않거나 제거되었습니다.

4. 콘텐츠 모델
- HTML5는 "인라인"과 "블록 레벨" 대신 새로운 콘텐츠 카테고리를 도입했습니다.
- 요소들의 허용된 자식 요소 규칙이 변경되었습니다.

5. API
- 새로운 API들이 도입되었습니다 (예: 미디어 요소 API, 오프라인 웹 애플리케이션 API, 편집 API 등).
- 기존 API들이 확장되거나 변경되었습니다.
- Document, HTMLElement 등의 인터페이스가 확장되었습니다.
- 일부 오래된 API들은 제거되거나 사용 중지되었습니다.

HTML5는 웹 개발자들에게 더 많은 기능과 유연성을 제공하며, 웹 애플리케이션 개발을 위한 강력한 도구들을 제공합니다. 이 문서는 HTML5와 HTML4의 주요 차이점을 개괄적으로 설명하고 있습니다.
