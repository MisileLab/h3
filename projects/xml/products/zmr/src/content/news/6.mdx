## 기능들

- 기능으로 가득찬 16진수 도구
- 파일의 내용을 파싱하고 강조하기 위한 사용자 정의 C++과 유사한 패턴 언어
- 테마 지원
- import와 export 지원
- 데이터 검사기
- 노드 기반 데이터 전처리
- 여러 다른 데이터 소스에서 로딩
- 데이터 검색
- 데이터 해싱 지원
- Diffing 지원
- 연동된 디스어셈블러
- 북마크
- 기능 많은 데이터 분석기와 시각화 도구
- YARA 지원
- 쉬운 도구들
- 내장된 콘텐츠 업데이터
- 현대적인 인터페이스
- 쉽게 시작 가능
