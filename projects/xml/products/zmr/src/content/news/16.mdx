## [Pretalx Talk at DevConf.cz 2024](https://pretalx.com/devconf-cz-2024/talk/W3AVCT)
이 문서는 GRUB 부트로더를 더 빠르고 안전한 Linux 기반 대안인 **nmbl** (부트로더 더 이상 없음)로 교체하려는 이니셔티브에 대해 설명합니다.  
이 프로젝트는 UEFI 시스템에서 EFI 스텁에 의해 직접 로드되는 부트로더로 Linux 커널 자체를 활용하려고 합니다.  
이 접근 방식은 전통적인 GRUB의 복잡성과 보안 취약점을 피하면서, Linux 커널의 고급 기능과 빠른 개발 주기를 활용합니다.  
문서는 이러한 접근 방식의 장점을 설명하고, 이 접근 방식에 대한 커뮤니티의 피드백을 구하고 있습니다.
